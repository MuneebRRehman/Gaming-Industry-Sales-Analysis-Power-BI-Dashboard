🎮 Gaming Industry Sales Analysis | Power BI Dashboard
📌 Project Overview

This project presents an interactive Power BI dashboard analyzing global video game sales data. The goal was to transform raw Excel data into meaningful business insights through data cleaning, modeling, and visualization.

The dashboard provides insights into genre performance, platform sales distribution, regional comparisons, and long-term sales trends within the gaming industry.

📊 Dataset Information

The dataset contains:

Game Name

Platform

Year of Release

Genre

Publisher

NA Sales

EU Sales

JP Sales

Other Sales

Global Sales

Sales values are represented in millions of units.

🛠 Tools & Technologies Used

Power BI

Power Query (Data Cleaning & Transformation)

DAX (Measures & KPIs)

Microsoft Excel (Data Source)

🎯 Key Insights

Certain genres dominate global sales.

Platform performance varies significantly across years.

Regional sales trends show strong differences between North America, Europe, and Japan.

A small number of top titles contribute heavily to total sales.
